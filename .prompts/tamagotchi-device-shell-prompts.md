# Nano Banana 2 Prompts for Tamagotchi Shell Support

## Prompt 1
Create a pocket-device UI chrome for a classic virtual pet game. The frame should feel tactile and nostalgic, with a screen window, chunky button area, subtle molded plastic shading, and space for a pet viewport plus status indicators. Use warm cream, muted purple accents, and playful toy-like proportions.

## Prompt 2
Design small icon concepts for virtual pet status states: sleep, sickness, poop, attention call, discipline, and clean-up. Keep the icons simple, readable at small sizes, and visually consistent with a handheld toy interface.

## Prompt 3
Create a set of UI backdrops and badges for a dinosaur pocket device. Include a screen border treatment, alert badge, and a few toy-like decorative elements that can sit behind the pet viewport without overpowering it.
