# Nano Banana 2 Prompts for Tamagotchi Sprite Generation

## Prompt 1: Egg Variants
Create a clean, toy-like tamagotchi egg sprite for a prehistoric virtual pet. The egg should feel classic and handheld-game friendly, with a transparent background, crisp pixel-art edges, a readable shell silhouette, and a small species-specific accent pattern. Keep the design centered in a square frame, avoid text, logos, and scenery, and preserve a deterministic color palette for the same species ID.

## Prompt 2: Hatchling Sprite
Create a compact pixel sprite of a hatchling dinosaur suitable for a tamagotchi UI. Keep the proportions chibi and readable at very small sizes, with bold silhouette separation, a transparent background, and no extra props. Preserve the species identity, but simplify details so the sprite stays crisp when scaled down.

## Prompt 3: Juvenile Sprite
Create a slightly larger and more energetic pixel sprite of the same prehistoric species in its juvenile stage. Use a transparent background, clean silhouette, clear species markers, and game-friendly readability. Keep the body shapes bold and consistent with a tiny device screen.

## Prompt 4: Adult Sprite
Create a mature tamagotchi sprite for the same species. The art should still read as pixel-friendly and compact, with a transparent background, a strong silhouette, and no scenery. Keep the same species identity and palette family across all stages so the evolution feels coherent.

## Prompt 5: Batch Rules
When generating a collection, keep every sprite in the same visual family: consistent outline weight, limited palette, transparent background, centered framing, and deterministic output for the same species ID. Do not add text, watermarks, or background props.
